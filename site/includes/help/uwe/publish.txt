uwe-publish 0.48.12
Publish a website

USAGE:
    uwe publish [FLAGS] <env> [project]

FLAGS:
    -e, --exec              Allow hook command execution
    -h, --help              Prints help information
    -s, --sync-redirects    Sync local redirects with remote
    -V, --version           Prints version information

ARGS:
    <env>        Publish environment
    <project>    Project path [default: .]
