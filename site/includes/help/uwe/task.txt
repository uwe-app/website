uwe-task 0.41.0
Utility tasks

USAGE:
    uwe task <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    alias              Manage site aliases (symbolic links)
    check-deps         Check project for local dependencies
    help               Prints this message or the help of the given subcommand(s)
    list-blueprints    List blueprints
    update-runtime     Update runtime assets
