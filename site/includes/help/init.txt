ht-init 0.18.3
Create a new project

USAGE:
    ht init [OPTIONS] [ARGS] [SUBCOMMAND]

FLAGS:
        --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -h, --host <host>                  Host name for the new project
    -l, --language <language>          Language for the new project
    -L, --locales <locales>            Set multiple languages (comma delimited)
    -p, --private-key <private-key>    Private key to use for SSH connections [env: HT_SSH_PRIVATE_KEY]

ARGS:
    <target>    Output directory for the project
    <source>    A blueprint source path or URL

SUBCOMMANDS:
    help    Prints this message or the help of the given subcommand(s)
    list    List available blueprints
